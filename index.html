HTML 

В своём телеграмм канале (ТЫК!) я написал что такое HTML, тег и атрибут и как создать HTML файл и начать работать с ним.
Как пишется тег? Тег пишется так:
<p> СТРОЧКА </p>(Тег "p" делает текстовый абзац , в теге "p" можно написать текст. Тег "p" парный тег, 
т.е он имеет открывающий тег <> и закрывающий тег . Есть и одинарные теги, т.е они имеют только 1 тег и это открывающий.)
Как пишется атрибут? Атрибут пишется так:
<p style="color: red;"></p> (Что делает атрибут "style" я с позволением Всевышнего Аллаха вам объясню когда 
я буду писать про CSS.)

Чтобы начать писать код нужно написать HTML структуру.

Сначала мы в среде разработки пишем:
<!DOCTYPER html> (Этот одинарный тег информирует браузеру о том какую версию HTML мы используем. Если не писать
его то браузер по идеи должен начать пытаться узнать используемую версию HTML. Когда мы пишем просто html это означает
что мы используем самую новую версию HTML.)

Далее мы пишем тег html:
<html> 
</html> (Этот парный тег информирует о том что это HTML документ. Внутри него надо  писать остальные теги.)

Далее мы пишем  тег head и тег body:
<head>
</head>
<body>
</body> (В теге "head" пишется код который не виден пользователю (есть исключение), а в теге "body" пишется код
который обычно виден пользователю)

В теге "head" пишут данные сайта(Пример: <meta name="author" content="Muhammad"> ), заголовок страницы (Чтобы дать 
заголовок страницы нужно написать в теге "head" парный тег <title> </title> , и между открывающим тего и закрывающим вы
можете написать заголовок страницы. Тег "title" единственный тег который пишется в теге "head" и даёт информацию, которую
пользователь может увидеть) и кодировка. 

В теге "body" пишут весь каркас сайта (тексты, изображения, таблицы и тому подобное).

HEAD:

Мета данные это данные которые предоставляют информацию о сайте.
<meta name="author" content="Muhammad">  - Имя автора
<meta name="keywords" content="HTML, CSS, JS, VUE"> - Ключевые слова
<meta name="description" content="FRONTEND WITH MUHAMMAD - Познавательный сайт!> - Описание страницы
- В  атрибуте "name" тип данных, а  в атрибуте "content" сами данные.

Чтобы браузер знал какую кодировку нужно написать вот это:
<meta charset="utf-8">
- Где  "charset=""" пишем название кодировки , в моём случае это utf-8

 Чтобы задать заголовок странице мы пишем:
 <title>ТУТ ЗАГОЛОВОК</title>


Также в теге "head" можно писать стили, но это мы разберём с позволением Всевышнего Аллааха когда
я буду писать о CSS.

BODY:

Заголовок:
<h1> САМЫЙ БОЛЬШОЙ ЗАГОЛОВОК </h1>
<h2> ЗАГОЛОВОК ПОМЕНЬШЕ </h2>
<h3> ЕЩЁ ПОМЕНЬШЕ </h3>
<h4> ЕЩЁ МЕНЬШЕ </h4>
<h5> И ЕЩЁ </h5>
<h6> САМЫЙ МАЛЕНЬКИЙ ЗАГОЛОВОК </h6>

Строки можно делать с помощью этих тегов:
<p> no </p>
<div> no </div>
<pre> no </pre>
<kbd> no </kbd>
<span> no </span>

Изображения можно добавить с помощью тега "img"
и его атрибута "src":
<img src="ТУТ УКАЗЫВАЕМ ПУТЬ К ИЗОБРАЖЕНИЮ">

Подключить файл с стилями можно так:
<link href="ПУТЬ К ФАЙЛУ">
-Так же подключаются и скрипты.

Указать ссылку на что-то можно так:
<a href="ССЫЛКА"> ТУТ МОЖНО УКАЗАТЬ НАЗВАНИЕ СТРАНИЦЫ </a>

Сделать список можно так:
<ol>
         <li>Купить тетрадь</li>
         <li>Купить хлеб</li>
         <li>Купить молоко</li>
</o>

"ol" Это нумерованный список.

Ещё можно так:
<ul>
         <li>Купить тетрадь</li>
         <li>Купить хлеб</li>
         <li>Купить молоко</li>
</ul>

"ul" Это список без нумерации.

Сделать текстовое поле можно так:
<form>
         <input type text="text">
</form>

Чтобы дать заголовок текстовому полю можно использовать тег:
<label>ЗАГОЛОВОК ТЕКСТОВОМУ ПОЛЮ</label>

Чтобы изменить шрифт текста можно использовать вот такие теги:
<strong>ЖИРНЫЙ ШРИФТ</strong>
<em>КУРСИВ</em>
<big>ДЕЛАЕТ ТЕКСТ БОЛЬШЕ</big>
<small>ДЕЛАЕТ ТЕКСТ МЕНЬШЕ</small>

Чтобы сделать кнопку можно использовать вот такой тег:
<button>НАЗВАНИЕ КНОПКИ</button>

Есть ещё кнопка чтобы отчистить текстовое поле:
<input type="reset" value="НАЗВАНИЕ КНОПКИ">

Таблицу можно сделать так:
<table>
        <tr>
                  <th>ЧТО-ТО<th>
        </tr>
        <tr>
                  <td>ЧТО-ТО</td>
        </tr>
</table>


Пример кода HTML:
<html>
             <head>
             </head>
             <body>
                          <h1>FRONTEND РАЗРАБОТКА</h1>
                          <h3>Это разработка внешней части сайта</h2>
                          <button>Я ПОНЯЛ!</button>
             </body>
</html>

Чтобы выделить части сайта можно использовать теги:
<header>
</header> - Шапка

<main>
</main> - Основная часть

<footer>
</footer> - Футер





Вам не нужно знать все теги и все атрибуты
Тут далекоооооо не все теги и не все атрибуты, но если вы знаете принцип написания кода , то вам в FRONTEND разработке вам
остаётся  только искать нужные вам теги и атрибуты.
